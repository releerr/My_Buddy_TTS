My Buddy：基于神经同步动态演化（NSDC）架构的新一代高自然度 TTS 引擎

My Buddy 是一款面向数字人、具身智能、多模态智能体、智慧城市生活服务场景设计的轻量化、高自然度神经语音合成引擎，完全基于自研架构实现。本项目提出 NSDC (Neural-Sync Dynamic Conformer) 混合架构，以深度语义理解驱动韵律生成与声学建模，从底层解决传统 TTS 发音机械、韵律僵硬、情绪缺失、长文本不连贯等行业痛点，为智能交互系统提供具备真人质感、情感对齐、低延迟、可本地部署的语音生成能力。

核心技术亮点
My Buddy 依托全链路自研模块构建语音生成体系，核心技术具备自主知识产权，整体架构轻量化、可端侧部署、高鲁棒性。
BERT-Brain 语义中枢基于增强版 RoBERTa-WWM-Ext 预训练模型构建，通过深度语义编码实现文本理解、停顿时长、重音位置、呼吸节点与情感倾向的一体化预测，将文本从字面符号转化为具备表达逻辑的语义特征，从根源提升语音自然度与情绪一致性。
NSDC 自研声学架构采用创新 Lite-Conformer 核心结构，融合全局自注意力建模与 1D 深度卷积局部特征提取能力，兼顾长文本语气连贯性与发音细节清晰度，实现语义特征到声学特征的精准高效映射，模型轻量化且推理速度优异。
极致自然韵律生成依托空洞卷积残差模块 ResidualConv1DBlock 实现，大幅扩大音频特征感受野，精准捕捉人声细微纹理、呼吸起伏与韵律动态，彻底消除合成语音的机械电音感，还原真人发声质感。
广播级高保真音质输出集成自研优化的 BigVGAN 生成对抗声码器，支持 44.1kHz 超高采样率波形生成，高频细节还原度高、人声纯净通透，可满足政务播报、数字人直播、康养陪伴、公共服务等专业场景音质要求。
零延迟可视化交互系统配套前后端分离的智能交互看板，支持实时文本输入、流式语音输出、梅尔频谱动态监控，可快速完成效果验证、场景演示与功能调试，全流程离线可用。

系统架构
项目采用文本分析 - 声学建模 - 波形合成三层解耦架构，模块独立可替换，全链路自研可控，支持本地离线推理。
文本分析层（The Brain）以 TextEncoder 与 ProsodyPredictor 为核心，基于 RoBERTa 提取 256 维高维语义特征，通过 1D CNN 预测头完成停顿时长、重音、呼吸点等韵律标签输出，实现从文本理解到韵律指令的精准生成，完成传统 TTS 无法实现的语境级理解。
声学建模层（The Body）以自研 NSDC Conformer 为核心，通过时长调节器完成文本特征与音频帧的精准对齐，经由轻量化卷积与注意力混合结构处理，最终解码输出 80 维梅尔频谱特征，平衡全局语气表达与局部发音精度。
波形合成层（The Voice）以 BigVGAN Vocoder 为核心，将梅尔频谱高效转换为高保真、高清晰度的自然语音波形，基于 GAN 结构实现低失真、高鲁棒性合成，实时率 RTF＜0.1，可满足端侧实时交互需求。

bash
运行

cd MyBuddy
pip install -r requirements.txt
模型训练
bash
运行
# 训练 BERT-Brain 语义韵律模型
python models/bert_brain/train.py

# 训练 NSDC 自研声学模型
python train_nsdc.py --config config/v4_latest.yaml

运行

# 启动前端可视化界面
cd frontend && npm run dev
